🎓 Student Management System (Python CLI Project)
📌 Overview

This is a command-line based Student Management System developed using Python. The project allows users to store, manage, and analyze student academic records efficiently using CSV file handling.

🚀 Features
Add new student records (Name, Roll Number, Marks)
Input validation for:
Names (only alphabets)
Marks (0–100 range)
Unique roll numbers
Automatic average calculation
Data stored in CSV file
Summary report generation
Displays:
Total students
Class average
Topper & lowest scorer
🛠️ Technologies Used
Python
CSV Module
File Handling
Exception Handling
📊 How It Works
User selects options from CLI menu
Inputs student details
Data is validated and stored in students.csv
Summary report is generated dynamically
📂 Project Structure
main.py → Main program logic
students.csv → Data storage file
